Crear repositorio
git init
//crear una carpeta .git

Comandos basicos
git status (ver en que estado esta)

git add (agregar archivo)
    -git add . (agregar todos los archivos que se indique)
    -git add ruta/a/directorio (agrega todos los archivos del directorio)

    Git commit -m "mensaje" (Le digo a Git cuál fue la modificación entre "comillas")

    Git commit -am "mensaje" (agarras todo lo modificado y guardas, mezcla el paso de add +-m)
    
    Git log (ver historial)
